## Usage

This project uses `pnpm`. If that is not installed, the easiest way to get it is to run
```
npm i -g pnpm
```

Download dependencies

```bash
pnpm i
```

Run Fortuna with:
```bash
pnpm start
```

Use `ctrl+C` to exit when done asking questions.
